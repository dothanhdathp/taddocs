# Package

## tree

Gói này sử dụng để xem các tệp dưới dạng cây thư mục:

```bash
pacman -Sy tree
```

## Make

```txt
pacman -S make
```

Hoặc

```bash
pacman -S mingw-w64-x86_64-make
```

## Cmake

- Tải trước `gcc toolchain` 
    ```bash
    pacman -S mingw-w64-x86_64-toolchain
    ```
- Tải gói `cmake`
    ```bash
    pacman -S mingw-w64-x86_64-cmake
    ```
