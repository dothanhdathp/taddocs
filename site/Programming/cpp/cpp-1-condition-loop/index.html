    Condition And Loop

    Condition

    Phép toán so sánh

    if

    Điều kiện xoay quanh biến bool và hàm if\ Biến bool cũng có thể hiểu là một số với mặc định: - true = 1 - false = 0

    Thử đoạn code sau:

    main.cpp
    int main()
{
    bool A = true;
    bool B = false;
    if(A) {
        std::cout << "Value of A: " << A; // Because A=true, code work in here
    }
    if(B) {
        std::cout << "Value of B: " << B; // code not run in here
    }
    return EXIT_SUCCESS; 
}

    Value of A: 1
    Ngoài ra, các số khác 0 cũng được tính là true.

    if/else

    Có một kiểu khác của if là if-esle. Nghĩa là nếu if sai thì vào trường hợp còn lại.

    main.cpp
    int main()
{
    bool A = false;
    if(A) {
        std::cout << "Value if A: " << A; // Because A=true, code work in here
    } else {
        std::cout << "Value else A: " << A; // code not run in here
    }
    return EXIT_SUCCESS; 
}

    Value else A: 0

    Phép toán so sánh

    So sánh bằng

    So sánh bằng dùng dấu ==

    main.cpp
    int main()
{
    int a = 1;
    int b = 2;
    if(a==b) {
        std::cout << "a = b";
    } else {
        std::cout << "a != b";
    }
    return EXIT_SUCCESS; 
}

    a != b
    Ngược lại, sử dụng != nếu muốn kiểm tra không giống.
    main.cpp
    int main()
{
    int a = 1;
    int b = 2;
    if(a!=b) {
        std::cout << "a != b";
    } else {
        std::cout << "a == b";
    }
    return EXIT_SUCCESS; 
}

    a != b

    and/or

    • Với phép so sánh và (AND), cả hai điều kiện so sánh đều phải đúng.
      • So sánh (AND) dùng dấu &&
    • Với phép so sánh và (OR), chỉ cần một điều kiện đúng.
      • So sánh (OR) dùng dấu ||
    • Có thể nhiều hơn hai vế, và phép so sánh có thể

    main.cpp
    int main()
{
    bool A = true;
    bool B = false;
    std::cout << A&&B << std::endl; // false
    std::cout << A||B << std::endl; // true
    return EXIT_SUCCESS; 
}

    0
1

    Phép đảo

    Phép đảo để đảo ngược kết quả. Ví dụ muốn kiểm tra với Điều kiện A không đúng.

    main.cpp
    int main()
{
    bool A = true;
    if(!A) {
        std::cout << A;
    }

    return EXIT_SUCCESS; 
}

    1

    Vòng lặp

    Vòng lặp điều kiện và Vòng lặp trong khoảng

    Vòng lặp trong khoảng

    • Vòng lặp trong khoảng là for. Vòng lặp sẽ kết thúc với điều kiện theo biến đầu i.
      • int i=0; : Biến i khởi tạo bắt đầu từ giá trị 0
      • i<10;: Điều kiện thoát là nhỏ hơn 10
      • ++i: Tăng thêm một giá trị sau khi hết vòng lặp
      • Chương trình chính thực hiện trong {}
        main.cpp
        int main()
{
    for(int i=0; i<10; ++i) {
        std::cout << i << std::endl;
    }
    return EXIT_SUCCESS; 
}

    Vòng lặp điều kiện

    Vòng lặp điều kiện sử dụng cú pháp while và thoát chỉ với 1 điều kiện. Cho đến khi nào điều kiện chưa được thoả mãn thì còn trong vòng lặp {}.

    main.cpp
    int main()
{
    int i=0;
    while(i<100) {
        // Do code in here
        ++i;
    }
    std::cout << i << std::endl;
    return EXIT_SUCCESS; 
}
    Chương trình thoát với điều kiện cho đến khi biến i đến giá trị 100 
    100
    Vòng lặp có sơ đồ như sau:

    Vòng lặp điều kiện cũng có thể được sử dụng cùng chức năng với for như sau:

    main.cpp
    int main()
{
    int i=0;
    for(;i<100;;) {
        // Do code in here
        ++i;
    }
    std::cout << i << std::endl;
    return EXIT_SUCCESS; 
}
